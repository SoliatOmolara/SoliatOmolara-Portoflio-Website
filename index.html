Projects

Below are some of the projects I have worked on.

Health Insurance Charges Prediction with Regression.

I used Linear regression as baseline model while I added the L1 and L2 regularization and Polynomial Features as an addition model to predict the health insurance of some individuals using features like gender, age, bmi, number of children, and smoking habits.
For this project, I performed basic data cleaning process to prepare the data, I visualize several relationship and corrrelation between the features, I used feature engineering to transform the categorical features, and finally I made use of GridSearchCV for hyperparameter tuning and for choosing the best parameters for the Linear models.

Airline Passenger's Satisfaction Prediction.

I made use of classification models like K-Nearest Neigbours, Logistic Regression, and Decision Tree to build a predictive model to predict whether passengers are satisfied with the airline services. I performed basic data cleaning and visualizations to prepare and show relationships of the features, I also analyzed the categorical variables and used one-hot encoding to transformed them.

Exploratory Data Analysis for Machine Learning.

I also work on the Data analysis to show to the relationships and correlations between all the features of the dataset using visualization and logical inferences.
